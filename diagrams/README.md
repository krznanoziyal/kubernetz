# Example Diagrams

| File | Format | Description |
|---|---|---|
| `example-microservices.xml` | draw.io XML | Microservices platform: api, auth, notification, postgres, redis, kafka, monitoring |
| `example-microservices.mmd` | Mermaid | Same architecture as a Mermaid graph |

## How to use

1. Open the KubeBlueprint UI at http://localhost:5173
2. Click **Get Started**
3. On the Editor page, drop `example-microservices.xml` into the upload zone OR paste the contents of `example-microservices.mmd` into the Mermaid tab
4. Review the inferred architecture graph
5. Click **Validate** to see best-practice checks
6. Click **Generate** to produce the full project scaffold
7. Click **Download ZIP** to get the generated repository
